# 💫 About Me:
I'm a software developer currently working at Tiny Dinner, with experience in web development through two years of internships as well as my current role.<br>My journey has been anything but linear, starting in civil engineering, pivoting to electrical engineering, and eventually discovering a passion for programming.<br><br>Throughout three internships at Molex and AMD, I worked in full-stack development, gaining hands-on experience with API design, UI/UX design, building internal tools, and technologies such as React, Tailwind, Node.js, PostgreSQL, and MongoDB.<br><br>My varied path has made me adaptable and eager to learn.<br><br>I am currently working on a GitHub CI analytics tool that ingests repository events and workflow runs to visualize pipeline reliability, performance, and failures over time. I hope to build a practical developer tool that gives teams clear visibility into their CI pipelines so they can detect failures faster and ship software more reliably.
